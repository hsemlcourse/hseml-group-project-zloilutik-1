## Описание задачи

Построение симуляционной системы тактических пошаговых боёв и генерация синтетических данных для дальнейшего анализа и машинного обучения.

В рамках проекта реализуется упрощённая, но структурно приближенная к оригинальной игре система боя, основанная на механиках Heroes of Might and Magic III. При этом цель не состоит в полном воспроизведении игры, вместо этого создаётся облегчённая модель боя, которая сохраняет ключевые игровые зависимости, но упрощает сложные игровые системы для возможности математического моделирования и дальнейшей обработки данных.

**Задача:** Классификация - предсказание исхода тактического боя

**Датасеты:** 

Датасеты были самостоятельно собраны, очищены и приведены к единому формату для использования в симуляционной модели

1. https://www.kaggle.com/datasets/zlata0slepts/heroes-of-might-and-magic-3-heroes
2. https://www.kaggle.com/datasets/zlata0slepts/heroes-of-might-and-magic-3-spells
3. https://www.kaggle.com/datasets/zlata0slepts/heroes-of-might-and-magic-3-skills/data
4. https://www.kaggle.com/datasets/zlata0slepts/heroes-of-might-and-magic-3-war-machines/data
5. https://www.kaggle.com/datasets/zlata0slepts/heroes-of-might-and-magic-3-units/data

**Целевая метрика:** Accuracy, F1-score

Accuracy/F1-score - для задачи классификации исхода боя
ROC-AUC - для вероятностного предсказания победителя

На текущем этапе метрики используются только концептуально, так как основной фокус проекта - построение корректной симуляционной среды и генерация данных.
